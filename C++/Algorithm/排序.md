# 一、插入排序

伪代码：
![[插入排序.png]]

C++实现：
```c++
template <class T>
void insert_sort(T* a, int size)
{
	for(int i = 1; i < size; ++i)
	{
		if(a[i] >= a[i - 1])
			continue;
		int pos;
		for(pos = 0; pos < i && a[pos] <= a[i]; ++pos);
		for(int j = i - 1; j >= pos; ++j)
		{
			int tmp = a[j];
			a[j] = a[j + 1];
			a[j + 1] = tmp;
		}
	}
}
```

插入排序的时间复杂度为$O(n^2)$

merge(int* a, int p, int q, int r)
{
	int len1 = q - p + 1, len2 = r - q;
	int arr1[len1 + 1], arr2[len2 + 1];
	for(int i = 0; i < len1; ++i)
		arr1[i] = a[p + i];
	for(int i = 0; i < len2; ++i)
		arr2[i] = a[q + i + 1];
	arr1[len1] = arr2[len2] = INT_MAX; // 设置一个哨兵，减少对特例的讨论

	{
		if(arr1[p1] <= arr2[p2])
		{
			a[i] = arr1[p1];
			++p1;
		}
		else
		{
			a[i] = arr2[p2];
			++p2;
		}
	}
}
