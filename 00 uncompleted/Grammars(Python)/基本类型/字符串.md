# 一、Basics

1. 在Python中，用引号括起来的都是字符串，不管是单引号还是双引号
2. 在一个字符串中可以直接包含撇号和引号，出现二义错误时，使用转义符`\`
```python
'I told my friend, "Python is my favorite language!"'
```
3. 拼接字符串：直接使用`+`（类似于C++中的string类）
4. 字符串可以右乘一个int类型的数，表示重复若干遍所形成的新字符串

1. `title()`：修改单词首字符大小写，返回一个str类型或LiteralString类型
```python
message = "hello python"
print(message.title())
```
此时将输出"Hello Python"

2. `upper()/lower()`：全部输出大写/小写，返回一个str类型或LiteralString类型
```python
message = "Hello Python"
print(message.upper())
print(message.lower())
```
此时将输出
```python
HELLO PYTHON
hello python
```

3. `rstrip()`：默认返回丢弃末尾空白（包括空格、换行符等）的字符串
```python
language = "python "
print(language)
print(language.rstrip())
print(language)
```
输出为（注意第一排和第三排末尾都有一个空格）
```python
python 
python
python 
```
当给出参数时，删去末尾所有参数中的字符(区分大小写)
```python
name = "Alice BobBobb"
print(name.rstrip("Bob"))
print(name.rstrip("b"))
print(name.rstrip("ob"))
print(name.rstrip("Bbo"))
```
输出
```python
Alice # 注意空格并未被删除
Alice BobBo
Alice BobB
Alice # 注意空格并未被删除
```

4. `lstrip()`：默认返回丢弃开头空白的字符串，用法同`rstrip()`
5. `strip()`：默认返回丢弃头尾空白的字符串，用法同`rstrip()`
