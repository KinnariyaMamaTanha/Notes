# 一、Grammar

1. 创建列表(使用中括号将元素括起来)
```python
List = [ele1, ele2, ……]
```
2. 当使用`print()`输出列表时，将打印列表的内部表示，**包括中括号**，对字符串而言，**包括引号**，如：
```python
fruit = ["apple", 'orange', 'banana']
print(fruit)
```
output
```python
["apple", 'orange', 'banana']
```

3. 使用小括号运算符访问列表元素，索引从0开始，当索引为-1时，将输出最后一个元素，如：
```python
fruit = ["apple", 'orange', 'banana']
print(fruit[-1])
```
output
```python
banana
```
同理，当索引为-k时，将输出倒数第k个元素

4. 同一个列表中可以存储不同类型的数据：
```python
List = ['elem', 1, 0.2, True, None]
```

# 二、Advanced

## 2.1 添加、删除元素

>*Python中的列表大多是动态的，因此可以修改、添加、删除元素*

1. ==`append()`==：在列表末添加元素
```python
fruit = []
fruit.append("apple")
fruit.append("orange")
print(fruit)
fruit.append("banana")
print(fruit)
```
output
```python
["apple", "orange"]
["apple", "orange", "banana"]
```

2. ==`insert()`==：在列表中插入元素
```python
fruit = ["apple", "orange"]
fruit.insert(0, "banana") # 把"banana"插入到索引0的位置
print(fruit)
```
output
```python
["banana", "apple", "orange"]
```

3. ==`del`语句==：在列表中删除元素
```python
fruit = ["apple", 'orange', 'banana']
del fruit[1] # 删除索引1处的元素，其余元素自动补齐
print(fruit)
```
output
```python
["apple", 'banana']
```

4. ==`pop()`==：当未指明索引时，删除列表末的元素**并返回其值**
```python
fruit = ["apple", 'orange', 'banana']
poped_element = fruit.pop();
print(poped_element)
print(fruit)
```
output
```python
banana
["apple", 'orange']
```

 当指明索引时，删除索引处的元素**并返回其值**
```python
fruit = ["apple", 'orange', 'banana']
poped_element = fruit.pop(1);
print(poped_element)
print(fruit)
```
输出
```python
orange
["apple", 'banana']
```

5. `remove()`：根据值删除第一个元素
```python
fruit = ["apple", 'orange', 'banana', 'orange']
fruit.remove('orange')
print(fruit)
```
输出
```python
["apple", 'banana', 'orange']
```

## 2.2 排序

1. `sort()`：将某列表永久排序，接受三个参数，并返回None值：
```python
sort(*, key:None = None, reverse: bool = False) -> None
```
```python
fruit = ['apple', 'orange', 'banana']
fruit.sort()
print(fruit)
```
输出
```python
['apple', 'banana', 'orange']
```

2. `sort(reverse = True)`：反向永久排序
```python
fruit = ['apple', 'orange', 'banana']
fruit.sort(reverse = True)
print(fruit)
```
输出
```python
['orange', 'banana', 'apple']
```

3. `sorted()`：对列表临时排序
```python
fruit = ['apple', 'orange', 'banana']
print(sorted(fruit))
print(fruit)
```
输出
```python
['apple', 'banana', 'orange']
['apple', 'orange', 'banana']
```
可以添加`reverse = True`参数
```python
fruit = ['apple', 'orange', 'banana']
print(sorted(fruit, reverse = True))
```

## 2.3 翻转列表

使用`reverse()`永久反转列表顺序
```python
num = [1, 2, 3]
num.reverse()
print(num)
```
输出
```python
[3, 2, 1]
```

## 2.5 range()

^2ce955

1. range(): return a list of integers(start defaults to 0)
```python
for val in range(1, 5)
	print(val, end = ' ')
for val in range(6)
	print(val, end = ' ')
```
output
```python
1 2 3 4 # 注意range()是左闭右开的
0 1 2 3 4 5 # 默认从0开始
```

2. create a list by using range()
```python
num = list(range(1, 5))
print(num)
```
output
```python
[1, 2, 3, 4]
```
`list()`: create an empty list by default, or with an argument(must be an iterable), and create a list, returning None

3. 指定步长：
```python
range(begin, end, n) # 以步长n来创建list
```
```python
for val in range(1, 11, 2)
	print(val, end = ' ')
```
output
```python
1 3 5 7 9
```

## 2.6 切片

1. 格式：
```python
List[begin : end : step]
```
2. 返回一个从List的begin到end（左闭右开）的部分且跨度为step所构成的列表
3. 当未给出时，begin默认从第一位开始，end默认在列表最后一位的再后一位
4. begin可以是负数，代表从倒数第几个开始
5. 切片可以用来复制数组：
```python
List2 = List1[:]
```
这时List2和List1是两个不同的列表，但是如果这样写：
```python
List2 = List1
```
则List2和List1是同一个数组（对比C++的数组名的意义是地址）
6. 切片还可以用于逆向输出数组
```python
print(List[::-1])
```

## 2.7 检查特定值

1. 使用关键字`in`:
```python
element in List
```
该句是一个bool表达式，当element在List中时，返回True；反之返回False

2. 使用关键字`not in`:
```python
element not in List
```
该句是一个bool表达式，当element不在List中时，返回True；反之返回False

## 2.8 集合

使用`set()`方法来获取一个没有重复元素的列表：
```python
num = [1, 1, 2, 4, 3, 2]
print(set(num))
```
output
```python
[1, 2, 4, 3]
```
