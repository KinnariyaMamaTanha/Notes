# 一、输出
## 1.1 print()

### 1.1.1 第一种版本
```python
print(*values: object, sep: str | None = " ", end: str | None = "\n", file SupportsWrite[str] | None = None, flush: Literal[False] = False) -> None
```
解释：
```python
print(value, ..., sep = ' ', end = '\n', file = sys.stdout, flush = False)
# Prints the values to a stream, or to sys.stdout by default
# Optional keyword argument:
# file: a file-like object (stream); defaults to the current sys.stdout
# sep: string inserted between values, default a space
# end: string appended after the last value, default a newline
# flush: whether to forcibly flush the stream
```
因此
1. 在默认情况下，使用print输出后会自动换行，当然也可以通过给end新的参数值来改变输出方式，如
```python
print("Hello, world", end = '\n\n')
print("Hello, world")
```
输出
```python
Hello, world

Hello, world
```

2. print()可以接受多个需要打印的参数(value)，同时可以通过修改`sep`的参数值来修改间隔每两个value之间的字符，如
```python
print('apple', 'banana', 'orange')
print('apple', 'banana', 'orange', sep = '&')
```
输出
```python
apple banana orange
apple&banana&orange
```

### 1.1.2 第二种版本

```python
print(*values: object, sep: str | None = " ", end: str | None = "\n", file _SupportsWriteAndFlush[str] | None = None, flush: bool)-> None
```
### 1.1.3 basics
可以使用类似于C语言中的`printf()`的语法来进行输出：
```python
c = 1
d = 2
print("The value of c is {:.lf}, and the value of d is {:.lf}".format(c, d))
```

# 二、输出
## 2.1 `input()`

接受一个参数，并输出该参数（也可以不接受），随后接收用户的输入并以**字符串形式**返回该值：
```python
num = input("Enter a number\n")
print(num)
```
output:
```python
Enter a number
12 # 用户输入
12 # 程序输出
```

## 2.2 `int()`

使用`int()`以获取数值输入（就是类型强制转换）
```python
num = input("Enter a number")
num = int(num)
```
或者
```python
num = int(input("Enter a number"))
```
