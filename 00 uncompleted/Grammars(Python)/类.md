1. 创建一个类：
```python
class Dog():  # 括号中是空的，说明需要从空白开始创建Dog类
	# 一次模拟小狗的简单尝试
	def __init__(self, name, age):  # 构造函数，名称固定，注意两边各有两条下划线
	# 初始化属性name和age
		self.name = name 
		self.age = age
	
	def sit(self): 
	# 模拟小狗被命令时蹲下
		print(self.name.title() + " is now sitting.") 
		
	def roll_over(self): 
	# 模拟小狗被命令时打滚
		print(self.name.title() + " rolled over!")
```

2. `__init__`函数：
注意`self`形参，当Python调用`__init__`函数创建Dog类对象时，将自动传入实参`self`，因此在创建Dog类对象时不需要传入，而只需传入其余实参即可。
且当每个与类相关联的方法被调用时，也会自动传入实参`self`
```python
mydog = Dog('Harry', 6) # 传入name, age实参, 而不需要self实参
```
