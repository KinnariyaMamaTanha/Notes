# 一、渐进记号

1. $f(n) = \Theta(g(n))$：g(n)是f(n)的渐进紧确界，即当n充分大时，f(n)和g(n)指向差一个常量系数（理解为等价无穷大）
2. $f(n) = O(g(n))$：g(n)是f(n)的渐进紧确上界（理解为高阶或同阶无穷大）
3. $f(n)=\Omega(g(n))$：g(n)是f(n)的渐进紧确下界（理解为低阶或同阶无穷大）
4. $f(n)=o(g(n))$：g(n)是f(n)的渐进非紧确上界（理解为高阶无穷大）
5. $f(n)=\omega(g(n))$：g(n)是f(n)的渐进非紧确下界（理解为低阶无穷大）
6. $f(n)=\Theta(g(n))$当且仅当 $f(n)=O(g(n))$ 且 $f(n)=\Omega(g(n))$
7. 匿名函数：
	1. 一个表达式中匿名函数的数目可以理解为等于渐进记号出现的次数，如在 $\sum^{n}_{i=1}O(i)$中，只有一个匿名函数，但在$O(1)+O(2)+ \ldots +O(n)$中却有n个匿名函数
	2. 当匿名函数出现在等式右边的时候，表示：无论怎样选择等号左边的匿名函数，总有一种办法来选择等号右边的匿名函数使等式成立。如 $2n^2+\Theta(n)=\Theta(n^2)$
