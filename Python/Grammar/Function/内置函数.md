| 名称             | 作用                    |
| ---------------- | ----------------------- |
| abs(x)           | 返回x的绝对值           |
| pow(x, y)        | 返回x的y次方            |
| round(x, ndigit) | 四舍五入的指定的精度    |
| help(func)       | 打印函数func的docstring |
| max(list)        | 返回序列中的最大值      |
| min(list)        | 返回序列中的最小值      |
| len(list)        | 返回序列长度                        |
