>*元组是不可修改的序列*

# 一、Basics

1. 创建元组：
```python
>>>1, 2, 3
(1, 2, 3) # 使用逗号分隔，自动生成元组
>>>1,
(1,) # 单个元素构成的元组也要加上都好
>>>(1, 2, 3)
(1, 2, 3) # 通常的构建方法
>>>(42)
42 # 仅仅加上括号不能创建元组
>>>tuple([1, 2, 3])
(1, 2, 3)
>>>tuple('Python')
('P', 'y', 't', 'h', 'o', 'n') # tuple函数接受一个序列，并转换为元组
```
`tuple`是一种类型，即元组，这里相当于使用了C++中的强制类型转换
