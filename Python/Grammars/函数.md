1. **格式**：
```python
def functionName(argument-list):
	……
```
2. **实参**：实参的相对位置不可更改（关键字实参除外）
3. **关键字实参**
```python
def describe_pet(animal_type, pet_name): 
	"""显示宠物的信息""" 
	print("\nI have a " + animal_type + ".") 
	print("My " + animal_type + "'s name is " + pet_name.title() + ".") 
	
describe_pet(animal_type='hamster', pet_name='harry')
```
此时两个实参的**相对位置可以改变**

4. **默认值**：可以和C++中一样为参数指定默认值，且有默认值的参数必须在最右边
5. **返回值**：使用`return`语句返回值
6. **重载**：Python中没有**重载**
7. **传递列表给函数**：可以向函数传递列表（实质上是列表地址），从而对列表进行修改；可以使用切片来防止函数修改列表
8. **接受未知个数的同类型参数**：
```python
def func(*arguments)
	……
func(1, 2, 3, 5, 7) # 此时arguments为一个以形参为参数的列表
```
也可以对关键字实参使用类似语法：
```python
def func(**arguments)
	……
```
