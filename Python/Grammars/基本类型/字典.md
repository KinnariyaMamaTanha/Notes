>*字典是一系列‘键—值’对*

1. 格式：
```python
alien = {'color': 'green', 'points': 5}
```
冒号前的为“键”，冒号后的为“值”(注意冒号前不能有空格)

2. 访问字典中的值;
```python
alien['color'] # 返回值green
```

3. 修改、添加键-值对
```python
alien['color'] = 'yellow'
alien['x_pos'] = 25
alien['y_pos'] = 0
print(alien)
```
output
```python
{'color': 'yellow', 'points': 5, 'x_pos': 25, 'y_pos': 0}
```

4. 删除键-值对
```python
del alien['x_pos']
del alien['y_pos']
```

5. 多行定义：
```python
favorite_languages = { 
	'jen': 'python', 
	'sarah': 'c', 
	'edward': 'ruby', 
	'phil': 'python'
	}
```
行首必需缩进4个空格

6. 遍历字典
```python
for key, value in alien.item():
	…… # key代表键，value代表值，同时遍历

for key in alien.keys():
	…… # 只遍历键
for key in alien:
	…… # 可以省略keys()方法

for value in alien.values()
	……
```
`keys()`方法返回键列表
`values()`方法返回值列表
