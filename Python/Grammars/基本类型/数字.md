# 二、Methods

1. `str()`：将数字转换为字符串
```python
print("Hello " + str(404) + "Not Found")
```

2. `type()`：返回参数的数据类型
```python
print(type(3))
print(type(3.0))
```
输出
```python
<class 'int'>
<class 'float'>
```

3. Python支持多重赋值
```python
x, y = 2, 3
```

4. 可以将int类的数乘上字符串类型的数：
```python
print(3 * 'a')
```
output
```python
aaa
```

5. 可以将数字的字符串强制转换为int型的数
```python
print(int('3')) # valid
# print(int('a'))  invalid
print(int('34.9')) # 不是四舍五入
```
输出
```python
3
34
```

6. `abs()`：返回绝对值
