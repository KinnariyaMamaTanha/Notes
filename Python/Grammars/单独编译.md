1. 引入其他文件：
```python
import file # file是.py文件名
file.func() # 使用file中的函数需要指明来源文件
```
也可以将文件名简写：
```python
import numpy as np
……
```
2. 导入特定函数
```python
from file import funcName # 此后不需要指明该函数的来源文件就可以使用
from math import sin
from operation import mul
```
导入所有函数：
```python
from file import *
```
3. 当函数头太长时，**回车并Tab两次**以分段
