### 几种初始化方法

1. `int a = 24;`
2. `int a(24);`
3. `int a = {24};`当大括号中为空时，初始化为0
4. `int a{24};`

>*后两种被称为列表初始化，更推荐使用

### 列表初始化

^504369

1. 列表初始化在自动转换时**不允许缩窄**，如不能把`float`型的数转化为`int`型的数 ^a3da02
2. 结构体用列表初始化很方便

### 初始化数组

1. 只有在定义数组时初始化数组
2. 初始化数组时如果没有标明数组元素个数，编译器将自动判断，如：
```c++
int num[] = {1, 4, 3, 2};//自动判定num中有四个元素
```
