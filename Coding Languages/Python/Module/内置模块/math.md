| 函数名/数据名 | 作用                            |
| ------------- | ------------------------------- |
| floor(x)      | 返回x的向下取整                 |
| ceil(x)       | 返回x的向上取整                 |
| sqrt(x)       | 返回根号下x                     |
| pi            | Pi的float近似数值               |
| pow(x, y, z)  | 返回x的y次方模z的结果，z默认为1 |
| e             | 自然常数e                       |
| comb(n, k)    | 返回组合数$C_n^k$               |
| `hypot(x, y)` | 返回x, y的欧几里得范数                                |
