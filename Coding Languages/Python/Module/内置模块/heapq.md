函数：

| 函数名               | 作用                    |
| -------------------- | ----------------------- |
| heappush(heap, x)    | 将元素x压入堆中         |
| heappop(heap)        | 从堆中弹出最小元素      |
| \_heapify_max        | 创建一个大根堆          |
| heapify(heap)        | 让列表具有堆特征        |
| heapreplace(heap, x) | 弹出最小的元素并压入x   |
| nlargest(n, iter)    | 以列表形式返回iter中最大的n个元素 |
| nsmallest(n, iter)   | 以列表形式返回iter中最小的n个元素 |
