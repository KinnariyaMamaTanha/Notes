1. `zip(list1, list2)` ^1a1622
```python
>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> zip(a, b)
<zip object in <some-address>>
>>> list(zip(a, b))
[(1, 4), (2, 5), (3, 6)]
```
当两个列表的长度不相同时，将按较短的列表的长度进行配对
